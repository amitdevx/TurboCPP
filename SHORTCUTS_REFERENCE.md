## ⚡ EFFICIENCY TIPS - COPY/PASTE MINIMIZATION

### Strategy 1: Use Find & Replace Instead of Manual Copy-Paste
```
Ctrl+H → Type old text → Tab → Type new text → Replace All
```
**Saves**: Multiple copy-paste operations

### Strategy 2: Block Selection for Larger Chunks
```
Shift+Click at start → Shift+Click at end → Ctrl+K (copy)
```
**Saves**: Repeated selection clicks

### Strategy 3: Bookmark Locations
```
Ctrl+B to mark → Navigate → Ctrl+B to jump back
```
**Saves**: Scrolling up/down repeatedly

### Strategy 4: Keyboard-Only Navigation
```
Ctrl+G → Enter line number → Enter
```
**Saves**: Mouse clicks and scrollbar usage

### Strategy 5: Select & Replace Pattern
```
Ctrl+A → Ctrl+H → Enter pattern → Replace
```
**Saves**: Manual find operations

## 🎓 LEARNING TIP

**Recommended Key Combinations to Master First:**
1. `Ctrl+F5` (Compile & Run) - Most used
2. `Ctrl+S` (Save) - Essential habit
3. `Ctrl+F` (Find) - Avoid scrolling
4. `Ctrl+H` (Find & Replace) - Bulk changes
5. `Shift+Arrows` (Selection) - Efficient copy/paste
6. `Ctrl+A` + `Ctrl+H` (Select All + Replace) - Power feature

**Daily Driver Shortcuts:**
- File: `Ctrl+N`, `Ctrl+O`, `Ctrl+S`, `Alt+X`
- Edit: `Ctrl+K`, `Ctrl+V`, `Ctrl+U`
- Compile: `Ctrl+F5`
- Debug: `F7`, `F8`

## ✅ COPY/PASTE ALTERNATIVE WORKFLOWS

### Workflow 1: Fastest Method
1. `Ctrl+F` → Search text
2. `Ctrl+K` → Copy result
3. `Ctrl+V` → Paste elsewhere

### Workflow 2: Multi-Copy (Advanced)
1. Select with `Shift+Arrows`
2. `Ctrl+K` (copy)
3. Navigate with `Ctrl+G`
4. `Ctrl+V` (paste)
5. Repeat from step 3

### Workflow 3: Replace Instead of Copy
1. `Ctrl+H` (Find & Replace)
2. Copy text in "Find"
3. Type replacement in "Replace"
4. `Replace All` button
